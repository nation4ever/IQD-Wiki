# تعلم HTML

HTML (HyperText Markup Language) هي اللغة الأساسية لبناء صفحات الويب.

## العناصر الأساسية

```html
<!DOCTYPE html>
<html lang="ar" dir="rtl">
  <head>
    <title>عنوان الصفحة</title>
  </head>
  <body>
    <h1>العنوان الرئيسي</h1>
    <p>فقرة نصية</p>
  </body>
</html>
```

## العناوين

```html
<h1>عنوان رئيسي</h1>
<h2>عنوان فرعي</h2>
<h3>عنوان أصغر</h3>
```

## الروابط والصور

```html
<a href="https://example.com">رابط</a> <img src="image.jpg" alt="وصف الصورة" />
```

## القوائم

```html
<ul>
  <li>عنصر أول</li>
  <li>عنصر ثاني</li>
</ul>
```
