# الواجهة الأمامية (Front-end)

الواجهة الأمامية هي كل ما يراه ويتفاعل معه المستخدم.

## التقنيات الأساسية

### HTML

لغة توصيف الصفحات، تحدد هيكل الموقع.

```html
<h1>مرحباً بالعالم</h1>
<p>هذه فقرة نصية</p>
```

### CSS

لغة التنسيق، تحدد شكل وألوان الموقع.

```css
h1 {
  color: blue;
  font-size: 24px;
}
```

### JavaScript

لغة البرمجة، تضيف التفاعل للموقع.

```javascript
document.querySelector("button").addEventListener("click", () => {
  alert("تم الضغط!");
});
```
