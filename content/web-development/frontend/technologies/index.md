# تقنيات الويب الأساسية

## شنو راح تلكى هنا؟
بهذا القسم، راح نفصل اللغات الأربعة اللي لازم كل مطور واجهات (Frontend Developer) يعرفها مثل اسمه.
هاي اللغات هي حجر الأساس. مستحيل تكدر تبني موقع حقيقي أو تستخدم Frameworks مثل React أو Next.js بدون ما تكون فاهم ذني الأساسيات بشكل قوي.

---

## خارطة الطريق (Roadmap)

امشِ بهذا التسلسل بالضبط، لا تطفر ولا تستعجل:

### 1. [HTML (الهيكل)](/web-development/frontend/technologies/html)
أول خطوة بمسيرتك. هنا راح تتعلم شلون تبني "عظم" الصفحة، من العناوين للصور للفورمات، وشلون المتصفح يفهم المحتوى.
* **[ادخل لصفحة HTML ←](/web-development/frontend/technologies/html)**

### 2. [CSS (التصميم)](/web-development/frontend/technologies/css)
ثاني خطوة. بعد ما بنيت الهيكل، راح تتعلم شلون تحوله إلى تصميم حلو، متناسق، وأهم شي "متجاوب" (Responsive) يشتغل على الموبايل والحاسبة.
* **[ادخل لصفحة CSS ←](/web-development/frontend/technologies/css)**

### 3. [JavaScript (المنطق)](/web-development/frontend/technologies/javascript)
ثالث خطوة وهي الأهم. اللغة البرمجية الفعلية اللي راح تنطي "الروح" للموقع وتخليه يتفاعل ويا المستخدم ويرسل ويستقبل بيانات.
* **[ادخل لصفحة JavaScript ←](/web-development/frontend/technologies/javascript)**

### 4. [TypeScript (المستوى الاحترافي)](/web-development/frontend/technologies/typescript)
الخطوة الأخيرة بالأساسيات. هي JavaScript بس "محمية" (Type Safe) ومرتبة أكثر. تعلمها راح ينقلك لمستوى المحترفين ويسهل عليك الشغل بالمشاريع الكبيرة.
* **[ادخل لصفحة TypeScript ←](/web-development/frontend/technologies/typescript)**

## نصيحة من "سينيور"
لا تحاول تتعلمهم كلهم بنفس الوقت. البرمجة مو سباق، هي تراكم خبرات.
1.  خلص **HTML** (تاخذ منك أيام قليلة).
2.  اضبط **CSS** وافهم الـ Flexbox (تاخذ أسابيع).
3.  تعمق بـ **JS** لأن هي الأساس لكلشي راح يجي بعدها (تاخذ أشهر).

من تضبط ذني الثلاثة، أي تكنولوجيا جديدة راح تنزل بالمستقبل راح تكون سهلة عليك.
