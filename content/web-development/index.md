# Web Development

## شنو يعني موقع ويب؟

ببساطة، الموقع هو **برنامج يشتغل بالمتصفح**.

بدل ما تنصب برنامج بحاسبتك وتنزله، الموقع هو مساحتك اللي تكدر توصللها من أي مكان بالعالم عن طريق الإنترنت. قبل جانت المواقع بس "للقراءة" (مثل الجريدة)، بس هسة بـ 2026 الموقع صار **تطبيق ذكي** (Web App) تكدر تسوي بيه كلشي: تشتري، تبيع، تصمم، وتدير شغلك بالكامل.

![برمجة المواقع](https://www.4cpl.tech/ast/uploads/2023/06/Web-Development.jpg)



## بيش نستخدم المواقع؟ (Use Cases)

المواقع مو بس "صفحة تعريفية"، هسة هي أساس البزنس:

- **متاجر إلكترونية (E-commerce):** تبيع منتجاتك للكل بدون ما تأجر محل فعلي.
- **أنظمة إدارة (SaaS):** برامج للمحاسبة، إدارة المخازن، والشركات.
- **منصات تواصل:** أماكن تجمع الناس، مثل السوشيال ميديا والمنتديات.
- **واجهة شخصية (Portfolio):** الـ CV مالتك ومعرض أعمالك حتى الشركات تشوف شغلك.

## أمثلة على مواقع قوية

حتى تكون عندك فكرة، هاي أنواع مختلفة للمواقع:

1.  **Amazon / ZainCash:** مواقع للبيع والشراء والمعاملات المالية.
2.  **Google Docs / Figma:** أدوات إنتاجية تشتغل بالمتصفح كأنها برنامج منصب.
3.  **YouTube / Netflix:** منصات لبث المحتوى والفيديوات.
4.  **ChatGPT:** تطبيقات ذكاء اصطناعي تفاعلية.
